# 🌈 Happy Holi 2026 Animation

An interactive **Holi Festival Greeting Website** with colorful UI, splash animations, confetti effects and personalized wishes.

## ✨ Features

- 🎨 Colorful Holi theme UI
- 🎊 Animated confetti celebration
- 🖌️ Splash color interaction
- 👤 Personalized name wishes
- 🎵 Background music option
- 📤 Share button
- 📱 Responsive modern UI

## 🎯 How It Works

1. User enters their name.
2. Clicks **Create Wish**.
3. A personalized Holi greeting appears.
4. User can trigger color splash animation.
5. Confetti and music create a festival vibe.
